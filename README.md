A simple Java application that calculates a user's Body Mass Index (BMI) based on height and weight inputs.

Features

Accepts user height and weight input
Calculates BMI automatically
Displays BMI category:
Underweight,
Normal Weight,
Overweight,
Obese,
User-friendly interface.

Technologies Used

*Java
*Object-Oriented Programming (OOP)

How to Run

*Download or clone the project.

*Open the project in your Java IDE (NetBeans, Eclipse, or IntelliJ).

*Run the application.

*Enter height and weight values.

*View the calculated BMI result.
